Features

  • Swimming fish animation that mimics the macOS app
  • Mouse avoidance behavior - fish swim faster when you get close
  • Wave-based swimming paths for natural movement
  • Customizable tasks, colors, and font sizes
  • Responsive design that works on mobile and desktop
  • Easy integration with just a few lines of code
  • Lightweight sprite-based graphics for fast loading

Easy Integration

Add the BrainFish widget to any website with just a few lines of code:

<!-- Include the CSS and JS files --> <link rel="stylesheet" href="css/brainfish-widget.css"> <script src="js/brainfish-widget.js"></script> <!-- Configure your tasks --> <script> window.brainfishConfig = { tasks: [ 'Finish project proposal', 'Review team feedback', 'Schedule client meeting', 'Update documentation' ], theme: 'blue', // blue, green, orange, red, purple fontSize: 16, // Font size in pixels speed: 50, // Swimming speed (pixels/second) avoidMouse: true // Enable mouse avoidance }; </script>

Or initialize it programmatically:

const widget = new BrainFishWidget({ tasks: ['Task 1', 'Task 2', 'Task 3'], theme: 'green', fontSize: 18, speed: 75 }); // Update tasks dynamically widget.updateTasks(['New Task', 'Another Task']); // Clean up when done widget.destroy();
